# ServiceNow Help Desk & Ticketing Lab 🎫

## Overview
This project demonstrates IT Help Desk operations using a **ServiceNow Personal Developer Instance (PDI)** accessed via the Brave browser alongside **Windows Server 2022**. The lab simulates a real-world password reset request from ticket submission through Active Directory remediation to incident resolution in ServiceNow.

---

## Environment

| Component | Details |
|---|---|
| Platform | ServiceNow Personal Developer Instance (PDI) |

| Domain / Instance | izaiahtech.lab |

## What Was Built
- ServiceNow PDI configured with users and assignment groups
- Pokémon faction groups mirrored from Active Directory
- Password reset incident ticket created and resolved end-to-end
- Cross-platform workflow between ServiceNow and Active Directory
